  • Without punctuation, the selector is just the name of the tag
  • [attribute] selects anything with (any value of) that attribute
  • Inside the braces {} modifies anything matching that selector
  • Rules are a series of property-value pairs

The Box Model

  • Margin: Space between boxes
  • Border: Styled boundary
  • Padding: Space from border
  • Content: Stuff inside the box
  • ---
  • px: Pixels
  • %: relative to parent
  • Measure from:
    top right bottom left

In this class, width (or height) = border + padding + content
